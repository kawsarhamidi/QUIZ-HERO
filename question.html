Question 1. difference between foreach map find and filter

forEach() is used when you want to perform an action on each element in the array, map() is used when you want to modify each element in the array and create a new array with those modified values, find() is used when you want to find a specific element in the array based on a certain condition, and filter() is used when you want to create a new array with only the elements that meet a certain condition.

Question 2. difference between local storage and session storage and cookies

localStorage and sessionStorage are used for client-side storage in web applications, while cookies are used for both client-side and server-side storage. localStorage and sessionStorage are similar in that they are both key-value stores that are accessible only within the domain and protocol where they were created, but they differ in terms of their persistence and lifetime. cookies are different in that they are sent from a website to the user's browser and can be accessed across multiple domains and protocols.
